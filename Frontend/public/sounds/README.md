# Notification Sounds

Thư mục này chứa các file âm thanh thông báo.

## Cần thêm các file sau:

1. `new-order.mp3` - Âm thanh cho Waiter khi có đơn mới
2. `kitchen-order.mp3` - Âm thanh cho Kitchen khi có đơn cần nấu
3. `notification.mp3` - Âm thanh thông báo chung (fallback)

## Gợi ý nguồn âm thanh miễn phí:

- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/

## Lưu ý:
- Nên dùng file MP3 có dung lượng nhỏ (< 100KB)
- Thời lượng ngắn (1-3 giây)
- Âm thanh rõ ràng, không quá chói
